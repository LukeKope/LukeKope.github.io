Kyrie Irving Vector Art

This is a personal project I recently completed using Adobe Illustrator. I completed it within two days using a reference image and the pen tool in illustrator. I made this art of my favorite player on my favorite team (Kyrie Irving of the Boston Celtics). The logo featured is Kyrie Irving's logo. I focused mostly on using the pen tool to create geometric outlines of the shadows on the jersey and body. The piece is meant to be a desktop background, so to account for the nav bar at the bottom of the screen I moved Kyrie above the bottom of the screen. As you can see from the final picture, everything was separated into layers so as to allow make the workflow neat, clean, and efficient.
